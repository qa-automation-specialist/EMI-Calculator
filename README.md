# EMI-Calculator
EMI Calculator
EMI CALCULATOR – JAVA PROGRAM

This is a simple console-based Java program that calculates the
Equated Monthly Installment (EMI) for a loan based on:

Loan Amount

Annual Interest Rate

Loan Tenure (in years)

This project is ideal for beginners learning Java and basic
financial calculations.

FEATURES

Accepts:

Loan amount

Annual interest rate

Loan tenure in years

Automatically converts:

Annual interest to monthly interest

Loan tenure from years to months

Calculates accurate EMI using the standard formula

Displays the monthly EMI amount

Simple, clean, and beginner-friendly code

TECHNOLOGIES USED

Java

Scanner class for user input

Math.pow() for power calculations

HOW TO RUN THE PROGRAM

Save the file as:
EMICalculator.java

Open a terminal or command prompt in the project folder

Compile the program:
javac EMICalculator.java

Run the program:
java EMICalculator

EMI FORMULA USED

EMI = [P × R × (1+R)^N] / [(1+R)^N − 1]

Where:

P = Loan Amount (Principal)

R = Monthly Interest Rate

N = Total Number of Months

EXAMPLE OUTPUT

Enter Loan Amount: 500000
Enter Annual Interest Rate (%): 8
Enter Loan Tenure (in years): 5
Your Monthly EMI is: 10138.20
